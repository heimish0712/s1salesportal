P1-19 오늘 할 일 태그 필터/다음액션/한 줄형 UI 개편

수정 요약:
- 미완료/전체/완료 옆에 태그 필터를 표시하며, # 기호 없이 전화/메일/견적/컨택 등으로 빠르게 조회.
- 할 일 행 상단의 기존 수동/컨택예정 분류 배지를 제거하고 그 자리에 태그 선택 UI 배치.
- 할 일 행은 작성시간(HH:mm), 할 일, 처리내용, 완료예정시간을 한 줄형 레이아웃으로 표시.
- 완료 체크 시 완료예정시간 대신 완료시간을 표시하고, 완료시간은 현재시각 자동 입력 후 수정 가능.
- 오늘 할 일 행의 고객 바로가기에서 컨택이력 버튼 제거, 상세/발송만 유지.
- 고객 컨택 이력 모달의 다음액션 입력 영역에 다음액션 태그 선택 기능 추가.
- 컨택이력_DB에 다음액션태그 컬럼 추가, 오늘 할 일에서 다음액션 태그를 읽어 표시/필터링.
